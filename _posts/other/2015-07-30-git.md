git命令
=============

- ####摘取分支上部分功能上线
*基于某个提交点构建分支：*
```shell
git checkout -b publish 9eed4b17
```
*合入主干的某一个commit：*
```shell
git cherry-pick 5a2623e6
```

- ####删除远程分支：
```shell
git push origin --delete publish
```

- ####撤销远程分支的某次提交：
```
git revert a50a0f90
git pull
```

- ####拉取远程分支：
```
git checkout origin/develop-pos -b develop-pos
```
